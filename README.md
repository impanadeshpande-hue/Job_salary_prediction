# Job_salary_prediction
## Dashboard Overview
The Job Salary Prediction Dashboard provides insights into employee salaries based on factors such as experience, education, job role, and location. The dashboard helps users analyze salary trends and understand the key factors that influence compensation.

## Technologies Used
The dashboard was built using the following tools and technologies:
📊 Power BI Desktop 
📁 Power Query
🧠 DAX (Data Analysis Expressions)
📝 Excel/CSV Dataset
📁 File Format

## Dataset
The dataset used in this project was obtained from Kaggle and contains employee information such as experience, education, job title, and salary.
Source: Kaggle

## Features/Highlights
✅ Data Cleaning and Preprocessing
    Handled missing values and duplicate records.
    Prepared data for analysis and modeling.
✅ Exploratory Data Analysis (EDA)
    Analyzed salary trends across different job roles.
    Identified relationships between experience, education, and salary.
✅ Interactive Dashboard
    Built a Power BI dashboard with KPI cards, charts, and slicers.
    Visualized salary distribution and employee insights.
✅ Data Visualization
    Created charts to understand salary patterns and trends.
    Compared salaries across departments, locations, and experience levels.
✅ Performance Evaluation
    Evaluated model accuracy using appropriate metrics.
    Compared predicted salaries with actual salaries.
